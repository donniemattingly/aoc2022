    execute(arg)

    Executes the instruction defined by the opcode at ip

    Returns a tuple of the new memory and the new instruction pointer. The new instruction pointer is found by incrementing the current instruction pointer by 1 + the number of parameters of this instruction. The instruction pointer can also be set by an instruction, and if it's value is :halt the program halts.

    execute(memory, atom)

    Executes the given intcode program in memory (a list of integers) and returns the memory after the program halts.

    This sets the instruction pointer to 0 initially.
